核心功能

--

自动识别任务

从对话上下文自动提取当前任务，不需要手动输入任何东西

--

实时更新

每轮对话自动更新任务描述，任务完成自动切换到 [DONE]

--

历史追踪

完成一个任务开始下一个时，上一个任务保留可见

--

零配置

两行命令安装，自动选择最佳摘要后端（Claude API / Ollama / 关键词）

4/5

两行安装，开箱即用

# Clone the repo
git clone https://github.com/lighthouse-strategy/claude-tab-tracking.git

# Install
cd claude-tab-tracking && ./install.sh

# Done! Open a new Claude Code session.
